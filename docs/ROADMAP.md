# Roadmap

## Short term

- Add small unit tests for utility functions in `src/chem_ml_utils.py`.
- Save selected notebook outputs as figures for README examples.
- Add benchmark comparison tables for descriptor, fingerprint, MLP and GNN models.

## Medium term

- Add scaffold-split evaluation to every predictive notebook.
- Add model cards for each trained model.
- Add a small Streamlit or FastAPI demo for molecule input and property prediction.

## Long term

- Add ChEMBL or ZINC pretraining examples.
- Add odour/perfume multi-label prediction when a suitable public dataset is available.
- Add molecular graph models with bond features, GIN, GAT, MPNN and Chemprop-style message passing.
