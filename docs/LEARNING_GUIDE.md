# Learning Guide

## Suggested order

1. Project 1 — RDKit basics.
2. Project 2 — descriptor regression.
3. Project 3 — fingerprint regression.
4. Project 4 — classification.
5. Project 5 — neural networks, GNN and DeepChem workflows.

## Core idea

Every molecular ML workflow needs four decisions:

1. What molecule representation is used?
2. What target is being predicted?
3. What model is trained?
4. How is generalisation checked?

The notebooks revisit these four questions repeatedly using different representations and model families.
